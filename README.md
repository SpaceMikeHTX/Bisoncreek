# Bison Creek Media Website

This is a complete static website built with HTML, CSS, and JavaScript.

## Files
- `index.html` — page structure and content
- `styles.css` — responsive styling
- `script.js` — mobile navigation and contact form behavior

## Open locally
Double-click `index.html`.

## Publish
Upload all three files to Netlify, Vercel, GitHub Pages, your web host, or a WordPress custom HTML environment.

## Before publishing
1. Replace `hello@bisoncreekmedia.com` with your real email address.
2. Replace the sample gradient portfolio panels with your photography.
3. Add your real social links.
4. Connect the form to Formspree, Netlify Forms, HubSpot, or another form service if you want submissions stored online.
5. Replace the placeholder initials logo with your Bison Creek Media logo.
